# Episodic Blueprint Architecture (EBA)

An autonomous AI engineering system that solves context rot, infinite loops, and hallucination.

## Architecture Layers

1. **Hybrid Memory & Context Engine** — Externalized state via markdown files and memory packets
2. **Blueprint Orchestrator** — Deterministic code interleaved with AI reasoning
3. **Episodic Thread Weaving** — Isolated worker threads returning compressed episodes
4. **Validation & Safety** — Multi-model voting, visual proof, and risk-threshold approvals
